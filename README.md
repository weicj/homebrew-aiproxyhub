# Homebrew Tap for AI Proxy Hub

Current packaged version: `0.3.1`

## Install

```bash
brew tap weicj/tap
brew install ai-proxy-hub
```

Or install directly without a separate `brew tap` step:

```bash
brew install weicj/tap/ai-proxy-hub
```

## Update

```bash
brew update
brew upgrade ai-proxy-hub
```
