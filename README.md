# Homebrew Tap for AI Proxy Hub

Current packaged version: `0.3.1`

## Install

```bash
brew tap weicj/aiproxyhub
brew install ai-proxy-hub
```

Or install directly without a separate `brew tap` step:

```bash
brew install weicj/aiproxyhub/ai-proxy-hub
```

## Launch

After installation you can start the app directly from the shell:

```bash
ai-proxy-hub
aiproxyhub
```

## Update

```bash
brew update
brew upgrade ai-proxy-hub
```
